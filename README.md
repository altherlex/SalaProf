# SalaProf - Jazz and Ruby together
Web crawler for audio program Sala dos Professores from Eldorado estation

## Call
```ruby
require 'SalaProf'
SalaProf::Info.new(1) # first page
```

## Tools
```ruby
# Gems
net/http
uri
hpricot

# Cmd
wget.exe # for download
```
